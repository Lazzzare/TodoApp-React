# Getting Started

![.gif](/src/assets/gifi.gif)

1. First of all you need to clone app repository from github:

```
https://github.com/Lazzzare/GithubUserAPI-React
```

2. Next step requires install all the dependencies.

```
npm install
```

3. Running the Project

```
npm run dev
```
